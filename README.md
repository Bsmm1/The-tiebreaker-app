# B Tiebreaker ⚖️

An AI-powered decision-making assistant that helps you break the tie on tough choices. Using Google's Gemini AI, it provides comprehensive analysis through pros/cons lists, SWOT analysis, and comparison tables.

![Project Preview](https://picsum.photos/seed/tiebreaker/1200/600)

## 🚀 Features

- **AI Analysis**: Leverages Gemini 3 Flash for deep reasoning and decision support.
- **Pros & Cons**: Weighted breakdown of advantages and disadvantages.
- **SWOT Analysis**: Strategic evaluation of Strengths, Weaknesses, Opportunities, and Threats.
- **Comparison Tables**: Side-by-side comparison of different scenarios or options.
- **The Verdict**: A final AI-generated recommendation based on the analysis.
- **Modern UI**: Built with React, Tailwind CSS, and shadcn/ui for a polished, professional look.
- **Responsive Design**: Fully functional on desktop, tablet, and mobile.

## 🛠️ Tech Stack

- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **AI**: [Google Gemini API](https://ai.google.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏁 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Google Gemini API Key (Get one at [AI Studio](https://aistudio.google.com/))

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   cd b-tiebreaker
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up your environment variables:
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```
